SmartBAS BMS - Annual Maintenance Renewal Key Pool
====================================================

50 pre-generated MAINTENANCE renewal keys. Each key extends the licensed
installation by 365 days from the later of (a) today or (b) the current
expiry date. Each renewal key may only be used ONCE across all machines.

Format: SMREN-XXXX-XXXX-XXXX-XXXX
(Distinct prefix from activation keys which use SMBAS-...)

Applied via: Start Menu -> SmartBAS BMS -> Renew SmartBAS License
          or: C:\SmartBAS\tools\renew-license.ps1 (run as Administrator)

Behavior:
  - Renewal extends the CURRENT expiry by 365 days.
  - Renewing early does NOT forfeit remaining days -- the new expiry is
    calculated from the existing expiry date, not from today.
  - Renewing after expiry calculates from today's date.
  - Each renewal key is single-use across ALL machines.
  - Applied renewals are logged at C:\SmartBAS\logs\renewal.log
    and in the 'renewalHistory' array in license.json.
