SmartBAS BMS - License Key Pool
================================

50 pre-generated license keys embedded in the installer binary.
Each key may be activated on exactly ONE machine. Once activated, the
key is permanently host-locked to that machine's MachineGuid and cannot
be reused elsewhere.

Format: SMBAS-XXXX-XXXX-XXXX-XXXX
Dashes are auto-inserted as the user types on the activation page.

KEEP THIS FILE PRIVATE.
Anyone with access to this list can activate SmartBAS BMS on any
machine (until that specific key is first activated, at which point
it becomes permanently locked to that machine's MachineGuid).

To add more keys:
  1. Edit the ValidKeys() function in SmartBAS_Setup.iss
  2. Bump SetArrayLength(K, N) and append K[N-1] := '...'
  3. Recompile both installer variants
  4. Add the new key(s) to this file
